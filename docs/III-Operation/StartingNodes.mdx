## Managing Services

To help administer an instance of Cardano Node running as a `systemd` service, use the following commands:
```
# Display the status of the service
sudo systemctl status cardano-node
# Restart the service
sudo systemctl reload-or-restart cardano-node
# Stop the service
sudo systemctl stop cardano-node
```

## Monitoring Cardano Node Log Files

Regular monitoring of log files is essential for determining how an instance of Cardano Node is running.

**To display and filter Cardano Node log files:**
```
# Show the most recent journal entries, and continuously print new entries as they are appended to the journal
journalctl --unit=cardano-node --follow
# Show entries appended to the journal since midnight the day before today
journalctl --unit=cardano-node --since=yesterday
# Show entries appended to the journal on the current day
journalctl --unit=cardano-node --since=today
# Show entries appended to the journal on July 29, 2025 between midnight and noon
journalctl --unit=cardano-node --since='2025-07-29 00:00:00' --until='2025-07-29 12:00:00'
```

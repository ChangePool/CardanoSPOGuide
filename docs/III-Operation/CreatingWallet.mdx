      <p class="pi">To create a new wallet using a seed phrase:</p>

        <li>Using a terminal window on the computer designated as the block-producing node in your stake pool configuration, type the following commands to download the latest release of <a href="https://github.com/cardano-foundation/cardano-wallet" target="_blank">Cardano Wallet</a>:

          </CodeBlock>

          <CodeBlock language="bash">
            {`tar -xvzf <ArchiveName>`}

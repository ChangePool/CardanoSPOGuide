# Markdown Style Guide

In Docusaurus, the following [admonitions](https://docusaurus.io/docs/next/markdown-features/admonitions) are available when using Markdown:

:::note
Maecenas elementum sollicitudin nunc ac malesuada.
:::

:::tip
Vivamus eu euismod dolor. Ut mi orci, ultricies at efficitur sit amet, faucibus quis leo. 
:::

:::info
Aliquam commodo ut ante eget sollicitudin. Donec lobortis, nisl quis dapibus lobortis, nisi nunc sodales felis, rhoncus vehicula est ipsum ac nisl.
:::

:::warning
Proin non metus sit amet purus sagittis commodo. Vestibulum sit amet vestibulum justo. Suspendisse id nibh commodo, malesuada sem at, ullamcorper diam. 
:::

:::danger
Sed sit amet fermentum eros. Pellentesque non nunc sit amet odio cursus rutrum quis vestibulum lectus. Cras erat metus, rutrum nec accumsan non, venenatis a risus.
:::

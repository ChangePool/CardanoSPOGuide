      <p><a href="https://terminada.io/" target="_blank">TerminadaPool</a> documents a <a href="https://github.com/TerminadaPool/libsecp256k1-iog-debian" target="_blank">procedure</a> explaining how to create a Debian (DEB) package to install the <code>libsecp256k1</code> library—which Cardano Node requires—on an air-gapped, offline computer.</p>
